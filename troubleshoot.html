üêå My PC is Slow - Diagnostic Guide

1
Check What's Using Your Resources
Easy
  1. Press Ctrl + Shift + Esc to open Task Manager
  2. Click "More details" if in simple view
  3. Click on CPU, Memory, or Disk columns to sort by usage
  4. Look for programs using high percentages (80%+)
  5. Right-click ‚Üí End Task on anything suspicious you don't need
‚ö†Ô∏è Common culprits: Chrome (memory hog), Windows Update, antivirus scans, cloud sync apps (OneDrive, Dropbox)
2
Disable Startup Programs
Easy
  1. Open Task Manager (Ctrl + Shift + Esc)
  2. Go to Startup apps tab
  3. Disable programs you don't need at startup
  4. Keep: GPU drivers, audio drivers. Disable: Spotify, Discord, Steam, etc.
  5. Restart your PC
‚úì Impact: Faster boot times and more available RAM
3
Free Up Disk Space
Easy
  1. Press Win + E to open File Explorer
  2. Check if your C: drive is nearly full (red bar = bad)
  3. Search "Disk Cleanup" in Start menu and run it
  4. Check all boxes and click "Clean up system files"
  5. Also check "Previous Windows installations" (can be 10+ GB)
‚ö†Ô∏è Important: SSDs need 10-20% free space to perform optimally. If nearly full, performance drops significantly.
4
Scan for Malware
Easy
  1. Open Windows Security (search in Start)
  2. Click Virus & threat protection
  3. Click Quick scan (or Full scan for thorough check)
  4. Also run Malwarebytes Free for a second opinion
‚úì Tip: Windows Defender is actually good now. No need for paid antivirus.
5
Check RAM and Storage Health
Medium
  1. Check RAM: Search "Windows Memory Diagnostic" ‚Üí Restart and check
  2. Check SSD/HDD: Download CrystalDiskInfo
  3. Look for "Health Status" - should say "Good"
  4. "Caution" or "Bad" means your drive is failing - backup now!
‚ö†Ô∏è Signs of dying SSD/HDD: Random freezes, slow file transfers, files becoming corrupted, increasing bad sectors
6
Fresh Windows Install (Nuclear Option)
Last Resort

If nothing else works, a fresh Windows install often fixes deep issues:

  1. Backup all important files to external drive
  2. Go to Settings ‚Üí System ‚Üí Recovery
  3. Click "Reset this PC" ‚Üí "Remove everything"
  4. Or do a clean install from USB for best results
‚ö†Ô∏è This will delete everything! Make sure you have backups of documents, game saves, and important files.

üìâ Game Stuttering - Fix Guide

1
Check Your Temperatures
Easy
  1. Download HWiNFO64 or MSI Afterburner
  2. Run your game with monitoring overlay enabled
  3. Check CPU and GPU temperatures
  4. CPU over 90¬∞C or GPU over 85¬∞C = thermal throttling (stutters)
‚ö†Ô∏è If overheating: Clean dust, replace thermal paste, improve case airflow, or get better cooling
2
Enable/Disable V-Sync Properly
Easy
  • V-Sync OFF = Best for competitive games (lower input lag)
  • V-Sync ON = Fixes screen tearing but adds input lag
  • Best solution: Use G-Sync (NVIDIA) or FreeSync (AMD) if your monitor supports it
  • Enable in NVIDIA Control Panel or AMD Software, then disable V-Sync in-game
3
VRAM/RAM Running Out
Medium

Stuttering when looking around or entering new areas? Likely VRAM or RAM issue.

  1. Check VRAM usage with MSI Afterburner overlay
  2. If VRAM is maxed out: Lower texture quality
  3. Check RAM usage in Task Manager while gaming
  4. If RAM is near 100%: Close background apps or upgrade RAM
‚úì Quick fix: Lower texture quality from Ultra to High - often saves 2-4GB VRAM with minimal visual difference
4
Shader Compilation Stutters
Easy

Stutters that go away after playing the same area? Shader compilation.

  • This is normal for first-time loading in many games
  • Some games have "Compile shaders" option in graphics settings - let it complete
  • Stutters should reduce after playing through areas once
  • Don't delete shader cache in GPU drivers (unless troubleshooting)
5
CPU Bottleneck Check
Medium
  1. Use MSI Afterburner to monitor CPU and GPU usage
  2. CPU at 100%, GPU below 90% = CPU bottleneck
  3. Solutions for CPU bottleneck:
    • Increase resolution (shifts load to GPU)
    • Enable graphics-heavy settings
    • Cap frame rate to reduce CPU load
    • Close background apps (especially browsers)
6
Background App Interference
Easy

Common causes of random stutters:

  • Windows Update - Check if update is downloading
  • Antivirus scan - Exclude game folders from scans
  • Cloud sync - Pause OneDrive, Dropbox, Google Drive
  • Discord overlay - Try disabling it
  • Browser with many tabs - Close Chrome/Firefox
  • Recording software - OBS, NVIDIA Shadowplay can cause stutters

üí• Games Crashing - Fix Guide

1
Update GPU Drivers
Easy

80% of game crashes are caused by outdated or corrupted GPU drivers.

  1. Download latest driver from NVIDIA or AMD
  2. Choose "Custom installation" ‚Üí Check "Clean install"
  3. Restart PC after installation
‚ö†Ô∏è If new driver causes issues: Use DDU (Display Driver Uninstaller) in Safe Mode to completely remove drivers, then install a slightly older version
2
Verify Game Files
Easy

Steam:

  1. Right-click game ‚Üí Properties ‚Üí Installed Files
  2. Click "Verify integrity of game files"

Epic Games:

  1. Click three dots on game ‚Üí Manage ‚Üí Verify

Battle.net:

  1. Click gear icon ‚Üí Scan and Repair
3
Disable Overclocks
Easy

Unstable overclocks are a common crash cause that's often overlooked.

  • GPU: Open MSI Afterburner ‚Üí Reset all settings to default
  • CPU: Enter BIOS ‚Üí Load default settings
  • RAM: Try disabling XMP/EXPO in BIOS temporarily
‚úì If crashes stop: Your overclock is unstable. Lower the settings or increase voltage slightly.
4
Run as Administrator
Easy
  1. Find the game's .exe file
  2. Right-click ‚Üí Properties ‚Üí Compatibility
  3. Check "Run this program as administrator"
  4. Also try "Disable fullscreen optimizations"
  5. Click Apply ‚Üí OK
5
Check Windows Event Viewer
Medium
  1. Press Win + X ‚Üí Event Viewer
  2. Go to Windows Logs ‚Üí Application
  3. Look for red "Error" entries around the time of crash
  4. Double-click to see details - often shows which module crashed
  5. Google the error message for specific solutions
6
Reinstall Visual C++ Redistributables
Medium

Many games need Visual C++ runtime libraries. Corrupted ones cause crashes.

  1. Download the All-in-One installer from TechPowerUp
  2. Extract and run install_all.bat as Administrator
  3. This installs all versions from 2005-2022
  4. Restart PC

üíÄ Blue Screen (BSOD) - Error Codes

WHEA_UNCORRECTABLE_ERROR
Hardware Error
Usually caused by: Unstable overclock, failing hardware, or driver issues
Fix: Reset overclock, update BIOS, check RAM with MemTest86, check SSD health
KERNEL_SECURITY_CHECK_FAILURE
Kernel Security Failure
Usually caused by: Corrupted drivers, RAM issues, or malware
Fix: Run SFC /scannow, update drivers, scan for malware, check RAM
IRQL_NOT_LESS_OR_EQUAL
Driver Conflict
Usually caused by: Faulty driver or hardware conflict
Fix: Update/rollback recent drivers, especially GPU and network drivers
PAGE_FAULT_IN_NONPAGED_AREA
Memory Error
Usually caused by: Bad RAM, corrupted page file, or driver issues
Fix: Test RAM with MemTest86, reset page file, update drivers
SYSTEM_SERVICE_EXCEPTION
System Service Error
Usually caused by: Incompatible driver or corrupted system file
Fix: Run SFC /scannow and DISM, update all drivers, check for Windows updates
CRITICAL_PROCESS_DIED
Critical Process Failed
Usually caused by: Corrupted system files or failing storage
Fix: Run SFC /scannow, check drive health, may need Windows repair/reinstall
VIDEO_TDR_FAILURE
GPU Driver Crash
Usually caused by: GPU driver crash, overheating, or unstable GPU overclock
Fix: DDU and reinstall GPU drivers, check GPU temps, reset overclock
MEMORY_MANAGEMENT
Memory Management Error
Usually caused by: Faulty RAM, driver issues, or corrupted system files
Fix: Run Windows Memory Diagnostic, reseat RAM, try one stick at a time
!
General BSOD Troubleshooting Steps
  1. Note the error code - Write it down or take a photo
  2. Check recent changes - New driver, software, or hardware?
  3. Run System File Checker: Open CMD as Admin ‚Üí sfc /scannow
  4. Run DISM: DISM /Online /Cleanup-Image /RestoreHealth
  5. Update all drivers - Especially GPU, chipset, and network
  6. Check for Windows updates
  7. Test RAM - Use Windows Memory Diagnostic or MemTest86
  8. Check temps - Overheating can cause BSODs

‚ö´ PC Won't Boot - Diagnostic Guide

1
No Power At All
Easy
  • Check power cable is plugged in (both ends)
  • Check PSU switch is ON (back of PC)
  • Try a different power outlet
  • Check if PSU fan spins when you press power
  • If nothing: PSU may be dead - test with PSU tester or different PSU
2
Fans Spin But No Display
Medium
  1. Check monitor is plugged into GPU, not motherboard
  2. Try a different display cable (HDMI/DP)
  3. Remove and reseat RAM sticks
  4. Remove and reseat GPU
  5. Try booting with one RAM stick at a time
  6. Clear CMOS (check motherboard manual)
  7. Listen for beep codes (if speaker connected)
3
Stuck on Logo / Boot Loop
Medium
  1. Try booting into Safe Mode:
    • Force shutdown 3 times during boot to trigger recovery
    • Or boot from Windows USB ‚Üí Repair ‚Üí Safe Mode
  2. In Safe Mode, uninstall recent drivers/software
  3. Run sfc /scannow from Command Prompt
  4. If caused by Windows Update: Roll back in recovery options
4
"No Boot Device" Error
Medium
  1. Enter BIOS (Del or F2 during startup)
  2. Check if your drive is detected
  3. If not detected: Check SATA/NVMe cable connections
  4. If detected: Set correct boot order (your drive first)
  5. If still failing: Drive may have failed - try connecting to another PC

üî• PC Overheating - Cooling Guide

1
Clean the Dust
Easy
  1. Power off and unplug PC
  2. Take it outside (dust will go everywhere)
  3. Use compressed air to blow out dust from:
    • CPU cooler fins and fan
    • GPU heatsink and fans
    • Case fans (front, back, top)
    • PSU vents (don't open PSU)
    • Dust filters
  4. Hold fans still while blowing to prevent damage
‚úì Expected improvement: 5-15¬∞C reduction in temps if it was dusty
2
Improve Airflow
Easy
  • Front fans: Should be intake (pulling air in)
  • Rear/top fans: Should be exhaust (pushing air out)
  • Remove unused drive cages blocking airflow
  • Manage cables - bundle them away from airflow path
  • Don't put PC in enclosed cabinet
  • Keep 6+ inches clearance around all vents
3
Replace Thermal Paste
Medium

Thermal paste degrades over time (2-5 years). Replacing it can significantly reduce temps.

  1. Remove CPU cooler (carefully)
  2. Clean old paste with isopropyl alcohol (90%+)
  3. Apply pea-sized amount of new paste to center of CPU
  4. Reinstall cooler (don't over-tighten, use X pattern)
  5. Good paste brands: Noctua NT-H1, Thermal Grizzly Kryonaut, Arctic MX-4
‚úì Expected improvement: 5-20¬∞C depending on how old/dried the paste was
4
Set Custom Fan Curves
Easy

Stock fan curves are often too conservative. More fan speed = lower temps.

  • GPU fans: Use MSI Afterburner ‚Üí Fan tab ‚Üí Custom curve
  • Case/CPU fans: Use BIOS or motherboard software (e.g., ASUS AI Suite, MSI Dragon Center)
  • Aim for fans to ramp up starting around 60¬∞C
  • Balance noise vs cooling based on preference
5
Upgrade Cooling (If Needed)
Advanced

If cleaning and paste don't help enough:

  • CPU: Upgrade to larger tower cooler or AIO liquid cooler
  • GPU: Repaste, or consider aftermarket cooler (Arctic Accelero)
  • Case: Add more fans, or get case with better airflow (mesh front)
  • Undervolting: Reduces heat while maintaining performance (see Optimization page)
