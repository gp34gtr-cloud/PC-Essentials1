Windows Optimization Tips

üéÆ
Enable Game Mode
High Impact
Steps:
1. Press Win + I ‚Üí Gaming
2. Click "Game Mode"
3. Turn it On

Prioritizes game processes and prevents Windows Update during gameplay.
‚ö°
High Performance Power Plan
High Impact
Steps:
1. Search "Power Plan" in Start
2. Select "High Performance"
3. Or run: powercfg -duplicatescheme 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c

Prevents CPU throttling during gaming.
üñ•Ô∏è
Disable Hardware GPU Scheduling
Medium Impact
Steps:
1. Win + I ‚Üí System ‚Üí Display
2. Click "Graphics"
3. "Change default graphics settings"
4. Toggle based on your results

Test both on/off - varies by system.
üîÑ
Disable Startup Programs
High Impact
Steps:
1. Press Ctrl + Shift + Esc
2. Click "Startup apps" tab
3. Disable unnecessary programs

Keep: GPU drivers, audio. Disable: Spotify, Discord, etc.
üé®
Disable Visual Effects
Low Impact
Steps:
1. Search "Performance" in Start
2. "Adjust appearance and performance"
3. Select "Adjust for best performance"
4. Re-enable "Smooth edges of screen fonts"

Reduces desktop GPU usage slightly.
üîî
Disable Notifications While Gaming
Medium Impact
Steps:
1. Win + I ‚Üí System ‚Üí Focus Assist
2. Turn on "When I'm playing a game"
3. Set to "Priority only" or "Alarms only"

Prevents notification popups during games.
üì∫
Set Correct Monitor Refresh Rate
High Impact
Steps:
1. Right-click Desktop ‚Üí Display Settings
2. Advanced display settings
3. Set refresh rate to max (144Hz, 240Hz, etc.)

Many people forget this after setup!
üõ°Ô∏è
Add Game Folder to Antivirus Exclusions
Medium Impact
Steps:
1. Windows Security ‚Üí Virus protection
2. Manage settings ‚Üí Add exclusion
3. Add your games folder (e.g., Steam, Epic)

Reduces stuttering from real-time scans.

Proceed with Caution

Overclocking can increase performance but may void warranties, cause instability, or damage hardware if done incorrectly. Always research your specific hardware and start with small increments.

üéÆ GPU Overclocking

  • Use MSI Afterburner (works with all GPUs)
  • Increase Core Clock by +50MHz, test stability
  • Increase Memory Clock by +100MHz, test stability
  • Monitor temperatures (stay under 85¬∞C)
  • Run benchmarks (3DMark, Unigine) to test
  • Typical gains: 5-15% more FPS
Get MSI Afterburner ‚Üí

üß† CPU Overclocking

  • Requires unlocked CPU (K/X series)
  • Done through BIOS/UEFI
  • Increase multiplier, not base clock
  • May need to increase voltage (carefully!)
  • Good cooling is essential (tower/AIO)
  • Stress test with Prime95 or Cinebench
Watch Guides ‚Üí

üíæ RAM Overclocking (XMP/EXPO)

  • Easiest overclock - just enable XMP/EXPO
  • Enter BIOS (Del or F2 at boot)
  • Find XMP (Intel) or EXPO (AMD) setting
  • Enable Profile 1 (usually fastest)
  • Save and exit - RAM now runs at rated speed
  • Check with CPU-Z to verify
Watch Tutorial ‚Üí

üîß GPU Undervolting

  • Reduces heat AND power with same/better performance
  • Use MSI Afterburner curve editor
  • Lower voltage while maintaining clock speed
  • Safer than overclocking
  • Great for laptops or hot GPUs
  • Can reduce temps by 10-20¬∞C
